# webapi_lesson

このリポジトリには、Web開発の基礎知識に関するグラフィックレコーディング風のインフォグラフィックと、クラウドコンピューティングおよびGoogleサービスに関する教育リソースが含まれています。

## コンテンツ概要

### 1. グラフィックレコーディング風インフォグラフィック
`graphic-recording-infographic.html`は、視覚的に魅力的な教育リソースとしてグラフィックレコーディングスタイルで設計されており、日本語で提供されています。以下の4つの主要トピックを扱っています：

- **初心者でもわかるHTML概要**：HTMLの定義と目的、基本構造、よく使われる要素、HTML5の特徴（セマンティック要素、マルチメディアサポート、グラフィックス、レスポンシブデザイン）
- **Webサイトの仕組み概要**：クライアント-サーバーモデル、フロントエンドとバックエンドの概念、3つの主要なフロントエンド技術（HTML、CSS、JavaScript）、ドメインとホスティングの説明
- **HTTP/HTTPS概要**：HTTPの定義と目的、リクエスト-レスポンスサイクル、主要なHTTPメソッド（GET、POST、PUT、DELETE）、ステータスコードとその意味、HTTPSとそのセキュリティ上の利点
- **REST API概要**：RESTアーキテクチャの説明、RESTの原則（ステートレス、リソース指向、統一インターフェース、キャッシュ可能）、RESTfulなURL設計の例、データフォーマット（JSON）、API認証方法（APIキー、OAuth、JWT）

### 2. クラウドコンピューティングとGoogleサービスの入門ガイド (バージョン1)
`gcp-cloud-computing-1.html`は、温かみのあるデザインで作成された教育コンテンツで、以下のトピックを扱っています：

- **クラウドコンピューティング入門**：概要、主なメリット、クラウドの種類（パブリック、プライベート、ハイブリッド、マルチクラウド）
- **IaaS、PaaS、SaaSの違い**：各サービスモデルの特徴と利用例の比較
- **Google Cloud Platform (GCP)について**：主なサービスカテゴリと特徴の解説
- **Google Maps Platform**：主なAPI、活用例、料金体系の変更（2025年3月1日以降）
- **無料枠について**：GCPの無料枠と活用のポイント
- **Routes API**：主な機能、実装方法、料金体系、効果的な活用のコツ

### 3. クラウドコンピューティングとGoogleサービスの入門ガイド (バージョン2)
`gcp-cloud-computing-2.html`は、モダンなMaterial Designスタイルで作成されたレスポンシブな教育コンテンツで、バージョン1と同様のトピックを取り上げていますが、ナビゲーションやレイアウトが改善されています。サイドバーメニュー、スムーススクロール機能、モバイル対応設計を特徴としています。

## デザイン特徴

### グラフィックレコーディングインフォグラフィック
- カードとセクションを使った視覚的に魅力的なレイアウト
- 日本語フォント（Yomogi、Zen Kurenaido、Kaisei Decol）
- 温かみのあるカラーパレット
- バブル、アイコン、装飾的な背景などのビジュアル要素
