# webapi_lesson

このリポジトリには、Web開発の基礎知識、クラウドコンピューティング、および各種分析ツールに関する教育リソースが含まれています。グラフィックレコーディング風のインフォグラフィックや詳細なチュートリアルを通じて、技術的な概念を視覚的に理解しやすい形で提供しています。

## コンテンツ概要

### 1. Web開発の基礎知識（グラフィックレコーディング風インフォグラフィック）

視覚的に魅力的な教育リソースとして、日本語で以下のトピックを説明しています：

- **初心者でもわかるHTML概要**：HTMLの定義と目的、基本構造、よく使われる要素、HTML5の特徴
- **Webサイトの仕組み概要**：クライアント-サーバーモデル、フロントエンドとバックエンドの概念、主要なフロントエンド技術
- **HTTP/HTTPS概要**：HTTPの定義と目的、リクエスト-レスポンスサイクル、HTTPメソッド、ステータスコード
- **REST API概要**：RESTアーキテクチャの説明、RESTの原則、RESTfulなURL設計、データフォーマット、API認証方法

### 2. クラウドコンピューティングとGoogleサービス入門ガイド

温かみのあるデザイン（バージョン1）とモダンなMaterial Design（バージョン2）の2つのバージョンで提供される教育コンテンツです：

- **クラウドコンピューティング入門**：概要、メリット、クラウドの種類
- **IaaS、PaaS、SaaSの違い**：各サービスモデルの特徴と利用例
- **Google Cloud Platform (GCP)概要**：主なサービスカテゴリと特徴
- **Google Maps Platform**：主なAPI、活用例、料金体系（2025年3月1日以降の変更含む）
- **GCPの無料枠**：活用のポイント
- **Routes API**：主な機能、実装方法、料金体系、効果的な活用のコツ

### 3. Microsoft Power BI リソース

- **Power BI導入概要**：ビジネスインテリジェンスツールの概要、Power BIとPower BI Desktopの違い、ライセンス構成、導入時の検討ポイント
- **Power BI Free入門チュートリアル**：インストール手順、インターフェース詳細ガイド、データ接続方法、視覚化の作成、ダッシュボード構築、レポート共有などの詳細な解説

### 4. SAS SQLオプティマイザプロジェクト

- SAS Proc SQLの_MethodとTree_オプションに関する詳細解説
- SQLオプティマイザの動作原理とクエリ実行計画の解析
- パフォーマンス最適化のためのSQL実行メソッドの理解

## デザイン特徴

### グラフィックレコーディングインフォグラフィック
- カードとセクションを使った視覚的に魅力的なレイアウト
- 日本語フォントの活用（Yomogi、Zen Kurenaido、Kaisei Decol）
- 温かみのあるカラーパレット
- バブル、アイコン、装飾的な背景などのビジュアル要素

### GCPガイド（バージョン1）
- カラフルなカードレイアウト
- 視覚的なアイコンと区切り
- リスト形式での内容提示
- 温かみのあるデザインテーマ

### GCPガイド（バージョン2）
- モダンなMaterial Designスタイル
- レスポンシブレイアウト
- スティッキーナビゲーション
- フィーチャーカードデザイン
- モバイルフレンドリーなメニュー
