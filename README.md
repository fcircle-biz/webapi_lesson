# webapi_lesson

このリポジトリには、Web開発の基礎知識に関するグラフィックレコーディング風のインフォグラフィックが含まれています。

## コンテンツ概要

このインフォグラフィックは、視覚的に魅力的な教育リソースとしてグラフィックレコーディングスタイルで設計されており、日本語で提供されています。主に4つのトピックを扱っています：

### 1. 初心者でもわかるHTML概要
- HTMLの定義と目的
- HTML文書の基本構造
- よく使われる要素
- HTML5の特徴（セマンティック要素、マルチメディアサポート、グラフィックス、レスポンシブデザイン）

### 2. Webサイトの仕組み概要
- クライアント-サーバーモデルの説明
- フロントエンドとバックエンドの概念
- 3つの主要なフロントエンド技術（HTML、CSS、JavaScript）
- ドメインとホスティングの説明

### 3. HTTP/HTTPS概要
- HTTPの定義と目的
- リクエスト-レスポンスサイクル
- 主要なHTTPメソッド（GET、POST、PUT、DELETE）
- ステータスコードとその意味
- HTTPSとそのセキュリティ上の利点

### 4. REST API概要
- RESTアーキテクチャの説明
- RESTの原則（ステートレス、リソース指向、統一インターフェース、キャッシュ可能）
- RESTfulなURL設計の例
- データフォーマット（JSON）
- API認証方法（APIキー、OAuth、JWT）

## デザイン特徴

このインフォグラフィックは以下の要素を使用しています：
- カードとセクションを使った視覚的に魅力的なレイアウト
- 日本語フォント（Yomogi、Zen Kurenaido、Kaisei Decol）
- 温かみのあるカラーパレット
- バブル、アイコン、装飾的な背景などのビジュアル要素
- コード例とチェックリスト

## 使用方法

インフォグラフィックを閲覧するには、任意の最新ブラウザでHTMLファイルを開いてください。
