Microsoft Copilot 入門チュートリアル

1. Microsoft Copilotとは

Microsoft Copilotは、Microsoftが提供する包括的なAIアシスタントです。以前のBing Chatをベースに発展し、検索機能、クリエイティブな文章作成、画像生成、コード作成支援など多様な機能を提供します。Windowsやウェブブラウザなど、様々なプラットフォームから利用できる汎用AIアシスタントです。

主な特徴

  • 自然言語でのインターネット検索と情報取得
  • 画像、文章、音楽などのクリエイティブコンテンツの生成
  • プログラミングコードの作成と修正の支援
  • 複雑な質問への回答と問題解決のサポート
  • Windowsの操作や作業の効率化支援
  • 画像認識や分析機能

2. 始め方

アクセス方法

Microsoft Copilotにアクセスするには、以下の方法があります：

  1. Microsoft Edgeブラウザから： サイドバーのCopilotアイコンをクリック
  2. Windows 11から： タスクバーのCopilotアイコンまたはWindows+Cキーで起動
  3. ウェブサイトから： copilot.microsoft.comにアクセス
  4. モバイルアプリから： iOS/AndroidのMicrosoft Copilotアプリを使用

ヒント： Microsoft Edgeブラウザを使用すると、ウェブページの内容に関連した質問ができるなど、より統合された体験が得られます。

基本的な使い方

  1. Copilotを起動します
  2. チャットボックスに質問や指示を入力します
  3. Enterキーを押すか送信ボタンをクリックします
  4. Copilotのレスポンスを待ちます
  5. 必要に応じて、フォローアップの質問や修正依頼を行います

3. データ保護とエンタープライズセキュリティ

プライバシーとセキュリティの基本

Microsoft 365 Copilot Chatは、ユーザーのプライバシーとデータセキュリティを重視して設計されています。以下がその主な特徴です：

項目 詳細
データの取り扱い チャットの内容はMicrosoftのプライバシーポリシーに基づいて処理されます。Microsoft 365 Copilotでは、ユーザーのテナント内のデータがテナント外に流出することはありません。
プライバシー制御 組織の管理者は、Copilotの使用範囲や対象となるデータソースを詳細に制御できます。
データアクセス Copilotは、ユーザーがアクセス権を持つ情報のみを処理します。アクセス権のないデータは利用されません。
トレーニングデータ 顧客データはCopilotのモデルトレーニングに使用されません。

注意： 機密情報や個人情報を扱う際は、組織のガイドラインに従ってください。Copilotが生成した情報は常に確認することをお勧めします。

エンタープライズ保護機能

Microsoft 365 Copilot Chat のエンタープライズセキュリティ

企業環境でのMicrosoft 365 Copilot Chatは、以下のエンタープライズグレードの保護機能を提供します：

保護機能 詳細
Microsoft Purview連携 情報保護、データガバナンス、コンプライアンス管理のための包括的なソリューションと統合
機密情報ポリシー適用 DLP（データ損失防止）ポリシーに基づいて機密情報の取り扱いを制限
コンプライアンス境界 規制要件に準拠したデータ処理と保管のための境界設定
監査とモニタリング Copilotの使用状況と操作の包括的な監査ログ記録
データ残留ポリシー 地理的なデータ保存場所の制御と国際データ転送の管理
エンドポイント保護 デバイスベースのアクセス制御とセキュリティポリシーの適用
認証・認可 多要素認証、条件付きアクセスポリシー、ゼロトラストセキュリティモデルとの統合

管理者向けエンタープライズ制御

Microsoft 365の管理者は、Purview管理センターを通じて以下のエンタープライズレベルの制御が可能です：

  • Copilotポリシー設定： 組織全体または特定のグループに対するCopilotの有効化/無効化
  • データソース制限： Copilotがアクセスできるデータソース（SharePoint、OneDrive、Exchangeなど）の制限
  • コンテンツフィルタリング： 特定のコンテンツタイプや機密レベルのデータに対するCopilotの利用制限
  • ユーザーアクセス管理： どのユーザーがCopilotを使用できるかのきめ細かな制御
  • コンプライアンスポリシー統合： 既存のDLPポリシーとの統合によるデータ保護
  • 高度な監査： Copilotの使用状況、生成されたコンテンツ、アクセスされたデータの詳細な監査
  • 地理的データ制御： データソバレンティ要件を満たすための地理的データ処理の制御

コンプライアンス認証

Microsoft 365 Copilot Chatは、以下のコンプライアンス認証・基準に対応しています：

  • ISO 27001, 27018（情報セキュリティ管理）
  • SOC 1, 2, 3（サービス組織の統制）
  • GDPR（EU一般データ保護規則）
  • HIPAA（米国医療保険の携行性と責任に関する法律）
  • FedRAMP（米国連邦リスク管理プログラム）
  • 各国・地域の規制要件に対応するコンプライアンスフレームワーク

ユーザーレベルのデータとプライバシー設定

ユーザーは、管理者が設定した範囲内で以下のプライバシー設定をカスタマイズできます：

  • 個人の会話履歴の管理と削除
  • Copilotが自分のコンテンツにアクセスする範囲の設定
  • Copilotに共有する個人情報の制限
  • 特定のプロジェクトや文書へのCopilotアクセスの無効化

4. 実用的なプロンプト例

情報検索のプロンプト

持続可能なエネルギーの最新動向について教えてください。特に日本での取り組みと今後の展望を知りたいです。

東京から大阪への新幹線と飛行機の所要時間とコストを比較してください。平日の移動を想定しています。

創作支援のプロンプト

「SDGsと企業の社会的責任」をテーマにした1000字程度のブログ記事を書いてください。具体的な事例も含めてください。

チームビルディングのためのオンラインワークショップのアジェンダを作成してください。所要時間は2時間、参加者は10名程度です。

プログラミング支援のプロンプト

Pythonでウェブスクレイピングを行うコードを書いてください。特定のニュースサイトから最新記事のタイトルと要約を取得したいです。

JavaScriptで簡単なTodoリストアプリを作るコードを書いてください。タスクの追加、完了、削除機能が必要です。

画像生成のプロンプト

「未来の東京の街並み」をイメージした画像を生成してください。空飛ぶ車や緑豊かな高層ビルがある風景が見たいです。

「和モダン」をテーマにしたインテリアデザインの画像を作成してください。リビングルームを想定しています。

学習支援のプロンプト

量子コンピューティングの基本概念を、コンピューターの専門知識がない人にもわかりやすく説明してください。

英語の敬語表現について教えてください。ビジネスメールで使える丁寧な表現と例文を10個紹介してください。

5. 効果的な使用のためのヒント

質問・指示の仕方

  • 具体的に： 曖昧な指示よりも具体的な指示のほうが良い結果を得られます
  • 目的を伝える： 何のために情報が必要かを伝えると、より適切な回答が得られます
  • 複数の質問を分ける： 一度に多くのことを尋ねるより、一つずつ質問すると良いでしょう
  • プロンプトの改良： 最初の回答が期待通りでなければ、質問を言い換えたり詳細を追加したりしましょう

プロのヒント： 「この情報をもう少し詳しく」「別の観点から説明して」などのフォローアップ質問を使うと、より深い情報が得られます。

モードの使い分け

Microsoft Copilotには主に3つのモードがあります：

  • バランスモード： 情報の正確性と創造性のバランスが取れたモード（デフォルト）
  • クリエイティブモード： より自由で創造的な回答が得られるモード
  • 精確モード： 事実に基づいた正確な情報を重視するモード

質問の内容や目的に応じて、適切なモードを選択しましょう。

制限事項を理解する

  • Copilotは常に完璧な回答を提供するとは限りません。生成された情報は必ず確認してください
  • インターネットから情報を収集するため、古い情報や不正確な情報が含まれる場合があります
  • 画像生成や創作的な内容には著作権や倫理的な考慮が必要です
  • 特に専門的な内容や数学的計算については、誤りがある可能性があります

6. よくある質問

Q: Microsoft Copilotは無料で使えますか？

A: 基本的な機能は無料で使用できますが、一部の高度な機能や利用制限の緩和には、Microsoft 365の有料サブスクリプションが必要な場合があります。

Q: Copilotが生成した画像や文章の著作権は誰に帰属しますか？

A: Microsoft Copilotによって生成されたコンテンツの使用権はユーザーにありますが、法的な解釈は地域によって異なる場合があります。商用利用の場合は利用規約を確認してください。

Q: オフラインでも使用できますか？

A: いいえ、Microsoft Copilotはクラウドベースのサービスであり、機能するためにはインターネット接続が必要です。

Q: 会話の履歴は保存されますか？

A: はい、デフォルトでは会話履歴が保存されます。ただし、設定からこれを無効にしたり、履歴を削除したりすることが可能です。

Q: 生成されたコンテンツが不適切だと感じた場合はどうすればいいですか？

A: Copilotの回答に問題がある場合は、フィードバックボタンを使用して報告することができます。また、プロンプトを変更して再試行することも効果的です。

Q: Copilotは何語に対応していますか？

A: Microsoft Copilotは多くの言語に対応していますが、英語が最も充実しています。日本語も含む主要言語でも十分な機能を提供しています。
