金融統計学における変数分類

1. 基本分類

1.1 質的変数と量的変数

分類 説明 金融例
質的変数 (定性的変数) カテゴリーや特性を表す変数 証券タイプ (株式、債券、デリバティブ)、取引所名、格付け (AAA, AA, A)
量的変数 (定量的変数) 数値で測定できる変数 株価、金利、取引量、ROE、PER

1.2 離散変数と連続変数

分類 説明 金融例
離散変数 個別の値をとる変数 取引回数、株式の保有数、デフォルト回数
連続変数 連続的な値をとる変数 収益率、金利、インフレ率、ボラティリティ

1.3 独立変数と従属変数

分類 説明 金融例
独立変数 (説明変数) 他の変数に影響を与える変数 金利、GDP成長率、原油価格、失業率
従属変数 (被説明変数) 他の変数から影響を受ける変数 株価、債券利回り、為替レート

2. 測定尺度による分類

2.1 名義尺度 (Nominal Scale)

カテゴリーを表すだけで、順序に意味がない変数

金融例:

  • 企業の業種分類（金融、IT、エネルギーなど）
  • 証券タイプ（株式、債券、ETF）
  • 取引市場（東証、NYSE、ナスダック）
  • 通貨種類（円、ドル、ユーロ）

適用統計量: モード（最頻値）、頻度分析

2.2 順序尺度 (Ordinal Scale)

順序に意味があるが、間隔に一定の意味がない変数

金融例:

  • 信用格付け（AAA, AA, A, BBB...）
  • 投資家のリスク許容度（低、中、高）
  • 市場センチメント指標（強気、中立、弱気）
  • ESG評価ランク

適用統計量: 中央値、四分位数、順位相関

2.3 間隔尺度 (Interval Scale)

間隔が一定の意味を持つが、比率に意味がない変数

金融例:

  • 株価指数値
  • 消費者物価指数（CPI）
  • 日付（取引日、決算日）
  • 気温データ（商品先物取引の分析用）

適用統計量: 平均値、標準偏差、相関係数

2.4 比率尺度 (Ratio Scale)

絶対的なゼロ点があり、比率が意味を持つ変数

金融例:

  • 株価、債券価格
  • 収益率、利回り
  • 取引量、売上高
  • PER、PBR、ROE、ROA
  • ボラティリティ、VaR
  • 金利、インフレ率

適用統計量: 幾何平均、変動係数、あらゆる統計的処理

3. 金融特有の変数分類

3.1 時系列変数とクロスセクション変数

分類 説明 金融例
時系列変数 時間経過に伴う変化を表す変数 日次株価、四半期GDP、月次インフレ率
クロスセクション変数 特定時点での複数対象の違いを表す変数 業種別PER、国別金利、企業間ROE比較
パネルデータ変数 時系列とクロスセクションの両方の特性を持つ変数 複数企業の複数期間財務データ

3.2 リスク・リターン変数

分類 説明 金融例
リターン変数 投資収益を表す変数 株式リターン、配当利回り、債券利回り
リスク変数 不確実性や変動を表す変数 ボラティリティ、ベータ、VaR、最大ドローダウン

3.3 ファンダメンタル変数とテクニカル変数

分類 説明 金融例
ファンダメンタル変数 本質的価値に関連する変数 EPS、BPS、FCF、ROE、PER、PBR
テクニカル変数 価格や出来高パターンに関する変数 移動平均、RSI、MACD、ボリンジャーバンド
マクロ経済変数 経済全体に関連する変数 GDP成長率、失業率、インフレ率、マネーサプライ

4. 統計モデルにおける変数の役割

4.1 ファクターモデル変数

分類 説明 金融例
共通ファクター 複数の資産に影響を与える要因 市場リスク、規模ファクター、バリューファクター
個別ファクター 特定の資産に固有の要因 企業固有リスク、イディオシンクラティックリスク

4.2 確率過程変数

分類 説明 金融例
定常過程変数 時間によって統計的性質が変化しない変数 多くの金利スプレッド、一部の相対価格
非定常過程変数 時間によって統計的性質が変化する変数 株価、為替レート、絶対価格レベル
確率的ボラティリティ変数 変動性自体が確率的に変化する変数 オプション価格のインプライドボラティリティ

5. 実務における変数変換

変換方法 説明 金融例
対数変換 非線形関係を線形化、分布の正規化 株価の対数リターン、対数株価
差分変換 非定常時系列の定常化 金利の一階差分、株価変化率
標準化 異なるスケールの変数を比較可能に Z-スコア化したPER、標準化収益率
正規化 変数を特定範囲に収める [0,1]範囲に正規化したリスク指標
