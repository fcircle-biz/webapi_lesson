Power BI 概要

Power BI は Microsoft が提供するビジネスインテリジェンスツールであり、 組織内の様々なデータソースを統合してインタラクティブな可視化を実現します。

Power BI と Power BI Desktop の違い

機能・特性 Power BI Service Power BI Desktop
主な用途 レポート・ダッシュボードの閲覧・共有 レポート・データセットの作成・開発
実行環境 クラウドベース（SaaS） PC にインストール（Windows）
データ処理 制限あり 高度なデータ変換・モデリング
コラボレーション 共有・アクセス管理・アラート 限定的（ファイル共有のみ）
導入形態 ブラウザベース アプリケーション
!

利用フロー： 一般的に Power BI Desktop でコンテンツを作成し、Power BI Service へ発行後にエンドユーザーと共有するワークフローが標準
